# 📚 MiInstitutoApp

MiInstitutoApp, is a mobile application developed in **Kotlin** using the **Android Studio** framework.
The app is designed to improve communication and organization between teachers and students in an educational environment.

## Overview

EduConnect aims to digitize and simplify key aspects of academic interaction by providing:

* **Real-time communication**
* **Cognitive stimulation through mini-games**
* **Event creation and management**

The platform creates a centralized digital space where teachers and students can interact efficiently and productively.

---

## Features

### Real-Time Chat

* Instant messaging between teachers and students
* Smooth and responsive communication
* Designed for academic discussions and support

### Cognitive Stimulation Mini-Games

* Educational mini-games designed to stimulate cognitive skills
* Can be used by both students and teachers
* Encourages learning through interactive experiences

### Event Creation & Management

* Create and organize academic events (e.g., excursions, activities, meetings)
* Digital management of event details
* Simplifies coordination and communication
